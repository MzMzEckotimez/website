What is Lua?

Lua is a powerful, fast, lightweight, embeddable scripting language. It has been used in many popular applications since its inception in 1993. It has an active community.

Why is it special?

Lua has a clean syntax, only 8 basic types (with LuaJIT adding a ninth), first class functions with multiple return values and varargs, a versatile data structuring type, full lexical scoping, light threads, iterators, inheritance, exceptions, function environments, built-in pattern matching, tail calls, and a one-function module system.

Is it free?

Yes. Lua, LuaJIT and luapower are all Open Source Software and can be used in both open source and commercial applications for free, without any paperwork.

What is LuaJIT?

LuaJIT is a tracing just-in-time compiler (JIT) for Lua. It's widely considered to be one of the fastest dynamic language compilers around, and it supports a variety of platforms, including Windows, Linux and OS X, Android, iOS, PS3, PS4 and XBox 360.

LuaJIT is fully compatible with Lua 5.1, and includes many features from Lua 5.2.

LuaJIT extends Lua with a powerful ffi interface that allows calling C function directly, without the need to write glue code.
